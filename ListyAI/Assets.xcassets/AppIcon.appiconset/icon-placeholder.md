# App Icon Placeholder

To add a custom app icon:

1. Create a 1024x1024 icon image
2. Use Xcode's Asset Catalog to add it here
3. Design suggestions:
   - Simple microphone icon with gradient (blue to purple)
   - Or a list/bullet point icon with wave form
   - Use modern, minimal design
   - Colors: Blue (#007AFF) to Purple (#5856D6) gradient

For now, the app will use the default iOS app icon.
